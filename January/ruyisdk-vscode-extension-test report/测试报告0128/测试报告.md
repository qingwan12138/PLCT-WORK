# ruyi版本更新提示冲突
## 环境信息

OS: Ubuntu 25.04
插件版本：[0.1.1-beta.1](https://github.com/ruyisdk/ruyisdk-vscode-extension/releases/tag/0.1.1-beta.1)

## 操作步骤
1.正确安装多个版本的 ruyi，并且在 VSCode 中的 ruyi 包管理器可以自由切换版本  
![多版本切换](image/多版本切换.png)

2.用旧版本打开 Visual Studio Code。

## 预期结果
ruyi会弹出新版本ruyi的更新提示，询问是否更新。

## 测试结果
以不同方法安装了多个版本的ruyi，比如最新版的ruyi和旧版的ruyi安装方式不一致，在使用较旧版本的ruyi时打开vscode时，系统仍会提示是否需要更新 ruyi 包管理器的最新版本以及是否需要进行更新操作。 
![ruyi更新提示冲突](image/ruyi更新提示冲突.png)
